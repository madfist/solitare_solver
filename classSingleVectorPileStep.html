Detailed Description

Class for representing steps in a game as a bitmask.

it holds the following information

  • card to move
  • pile to move from
  • pile to move to
  • turning up the card on top
  • doing a stock move
  • original card position
  • new card position
  • weight of step (for heuristic solving)

  • weight
